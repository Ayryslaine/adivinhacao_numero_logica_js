# Lógica de Programação: Mergulhe em programação com JavaScript

Este repositório contém o projeto prático desenvolvido durante o curso de **Lógica de Programação: Mergulhe em programação com JavaScripto** da Alura. O foco principal foi consolidar os fundamentos da programação utilizando JavaScript para criar uma aplicação interativa.

## 🎯 Objetivo do Projeto
Desenvolver um jogo de adivinhação de números, onde o usuário interage com o navegador para tentar descobrir o número secreto gerado pelo sistema.

## 📝 Conteúdo Estudado
O curso foi dividido em 5 módulos principais, abrangendo:

1.  **Iniciando com JavaScript:** Configuração do ambiente, exibição de alertas e manipulação de variáveis.
2.  **Condicionais e Concatenação:** Implementação de estruturas `if/else`, uso de comentários e template strings.
3.  **Loops e Tentativas:** Controle de fluxo com o laço `while` e contagem de tentativas.
4.  **Boas Práticas:** Uso de operadores ternários, interrupções com `break` e geração de números aleatórios com `Math.random()`.
5.  **Desafios Práticos:** Resolução de problemas reais para fixação do conteúdo.

## 🛠️ Tecnologias Utilizadas
* **JavaScript:** Lógica principal do jogo.
* **HTML5 & CSS3:** Interface e estilização básica do projeto.
* **Git & GitHub:** Versionamento de código.
* **VS Code:** Ambiente de desenvolvimento.
